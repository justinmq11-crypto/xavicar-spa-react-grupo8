# XAVICAR SPA – Avance 70%

Segundo avance de la transformación del sitio de XAVICAR en una aplicación SPA con React, Vite y React Router.

## Incluido en este avance

- Todas las rutas principales.
- Menú responsive con `NavLink` y ruta activa.
- Componentes reutilizables.
- Catálogo completo desde `src/data/products.js`.
- Tarjetas creadas con `.map()` y props.
- Ruta dinámica `/producto/:id`.
- Lectura del parámetro mediante `useParams`.
- Página de detalle de producto.
- Página de servicios y promoción.
- Video original.
- Página de ubicación con mapa y horarios.
- Página de contacto con estructura visual.
- Página 404.
- Diseño responsive avanzado.

## Pendiente para el proyecto final

- Convertir el formulario en un componente controlado con `useState`.
- Agregar validaciones y mensajes de error.
- Confirmación visual de envío.
- Revisión final de accesibilidad.
- Pulido final de textos y documentación.

## Ejecutar

```bash
npm install
npm run dev
```
